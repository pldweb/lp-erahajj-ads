EH Starter

775 000/Bulan
1.625 User & Transaksi
Pesan Sekarang

Paket EH Starter menyediakan penyimpanan file hingga kapasitas 2 GB. Adapun contoh file diantaranya sebagai berikut :

  • File dokumen administasi (foto/scan KTP, foto/scan KK, dsb)
  • File bukti pembayaran transaksi
  • File gambar galeri paket
  • Dan lain sejenisnya

Seluruh file yang diupload ke sistem Erahajj akan dikompres tanpa mengurangi kualitas aslinya, sehingga memungkinkan travel Anda untuk menyimpan ratusan hingga ribuan file dalam sistem

Kapasitas user & transaksi yang disediakan dalam paket EH Starter adalah sebanyak 1.625. Artinya, paket EH Starter dapat menyimpan hingga 1.625 data transaksi, dan 1.625 data user

Data transaksi adalah transaksi seperti transaksi : umrah, haji, wisata, visa, hotel, land arrangement, dan lain sebagainya. Adapun data user adalah data : operator, agen, sales, dan jamaah yang tersimpan di sistem

Sebagai contoh, terdapat satu transaksi umrah dengan pemesan keluarga sejumlah 5 orang, maka dihitung sebagai : 1 transaksi dan 5 user (jamaah)

Data yang dihitung merupakan keseluruhan data yang tersimpan di sistem, bukan data secara periodik (bulanan, tahunan)

Apakah ada tutorial penggunaan sistem?
Erahajj menyediakan media pembelajaran untuk implementasi sistem yaitu :
  • Erahajj Learning Management System
    E-Learning berupa video dan panduan tertulis yang terdapat di masing-masing sistem Anda
  • Erahajj Training Center
    Pelatihan penggunaan sistem secara offline & online (hybrid) secara langsung dan gratis. Training offline berlokasi di kantor Erahajj, dan online melalui melalui Zoom atau Google Meet
  • Training Online Eksklusif
    Pelatihan penggunaan sistem secara eksklusif melalui Zoom atau Google Meet
  • Training Onsite Eksklusif
    Pelatihan penggunaan sistem secara eksklusif langsung di kantor Anda
Apakah saya bisa upgrade & downgrade paket sewaktu-waktu?
Tentu. Kami akan menghitung ulang sisa pembayaran sesuai dengan sisa waktu berlangganan dan selisih harga paket yang sedang digunakan ke paket yang akan diambil nantinya.
Apakah saya bisa migrasi data dari sistem lain?
Bisa. Anda bisa konsultasi terlebih dahulu dengan Travel System Consultant kami untuk proses migrasi data tersebut.
Bagaimana jika saya tidak lagi berlangganan Erahajj?
Anda tidak bisa mengakses ke dalam sistem lagi jika sudah tidak berlangganan. Anda bisa download data laporan maupun transaksi dalam bentuk file excel sebelum putus berlangganan, sehingga Anda masih memiliki arsip data untuk digunakan kedepannya.
